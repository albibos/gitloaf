Gitloaf

Gitloaf is an open-source nodejs powered CDN that delivers files from github with their correct Content-Type headers, similarly to other cdns such as githack. You can see a list of supported file types here

Current version: BETA 1.0.0

In beta versions, some or many games may not work. These issues will be fixed in later versions.
